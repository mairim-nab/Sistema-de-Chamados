# Sistema de Chamados – Front-end

Interface de gerenciamento de chamados desenvolvida para praticar lógica de front-end,
manipulação do DOM e regras de negócio com JavaScript puro.

## Funcionalidades
- Alteração de status do chamado (Aberto, Em andamento, Encerrado)
- Histórico automático de interações
- Envio de mensagens
- Encerramento de chamado
- Avaliação do atendimento com estrelas e comentário

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla JS)
